LinkedIn API Integration Setup

Configure your LinkedIn API credentials to fetch real analytics data

LinkedIn API not configured
1 Get LinkedIn API Credentials
  1. Go to LinkedIn Developer Portal
  2. Create a new app and select "Sign In with LinkedIn" and "Marketing Developer Platform"
  3. Add your app's authorized redirect URLs
  4. Get your Client ID and Client Secret
2 Generate Access Token

Use this OAuth 2.0 flow to get your access token:

https://www.linkedin.com/oauth/v2/authorization?
response_type=code&
client_id=YOUR_CLIENT_ID&
redirect_uri=YOUR_REDIRECT_URI&
scope=r_organization_social+rw_organization_admin+r_organization_follow

Then exchange the authorization code for an access token.

3 Configure API Credentials
Your LinkedIn OAuth 2.0 access token with organization permissions
Your LinkedIn company page ID (e.g., 109619030)
Test API Connection
